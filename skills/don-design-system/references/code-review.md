# Code Review op basis van NL Design System conventies

Voer een code review uit op de aangeleverde code. Controleer elk relevant onderdeel hieronder. Geef per bevinding aan:
- 🔴 **Overtreding** — code voldoet niet aan een conventie (met uitleg en correctie)
- 🟡 **Aandachtspunt** — mogelijk probleem, context-afhankelijk
- ✅ **Goed** — code voldoet aan de conventies

Vermeld alleen de gebieden die relevant zijn voor de aangeleverde code. Als er niets te melden is, zeg dan waarom de code voldoet.
